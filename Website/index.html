Enhanced Dock Experience

Smart Window Preview

Instantly preview all windows when hovering over Dock icons. Direct window selection replaces macOS's cycle-through approach, with intelligent window state memory and restoration.

Advanced Window Controls

One-click window snapping to left/right, maximize with multi-display support, and center positioning. Quick-hide active windows and terminate apps with double-click.

Multi-Display Magic

Seamlessly move windows between displays with visual indicators. Smart window management across screens with instant position memory and restoration.

Modern UI Integration

Beautiful bubble-style previews with smooth animations. Automatically adapts to light/dark mode with intuitive status indicators for window states.

Intelligent App Management

Smart Finder integration with desktop window handling. Single-click to hide active apps, remembering window positions for perfect restoration.

Native Performance

Optimized batch processing for window operations. Deep integration with macOS accessibility features ensuring smooth, native-like performance.
